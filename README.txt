Чтоб запустить скрипты нужно:
1) Получить credentials. Инструкция: https://developers.google.com/workspace/guides/create-credentials
2) Назвать этот файл creds.json и поместить его в рабочую директорию.
3) Выполнить консольную команду "pip install -r requirements.txt"
4) В файле "database.py" в 7, 8, 9 строках внести вашу информацию.
